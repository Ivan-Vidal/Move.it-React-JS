# Move.it-React-JS


## Descrição

<p>
  Projeto desenvolvido na semana de 22/02/2021 até 26/02/2021. Sediado pela Rocketseat e ministrada pelo CTO Diego Fernandes.
  O projeto consiste em construir uma aplicação para melhorar a organização de tempo de foco em estudos ou tarefas em frente ao computador. Utilizando os conceitos do sistema POMODORO, disponibilizando um periodo de tempo para foco total(nesse projeto será 25 minutos), e após, alguns minutos para fazer qualquer outra tarefa longe do computador. Você ganhará experiência e poderá subir de nível.
</p>


## Tecnologias utilizadas

- React JS;

- Next JS;

- Node Js;

- Yarn;

- TypeScript;
